collections framework
[
    collection interface
        - list (subinterface of collection and iterator)
            - ordered (can be accessed with index)
            - duplicate o
            - ex. ArrayList, Vector, LinkedList

        - set (subinterface of collection and iterator)
            - unordered (cannot be accessed with index)
            - duplicate x
            - ex. HashSet

        * map?(map is not a subinterface of collection and iterator)
            - key value pair
            - ordered
            - no duplicate keys
            - HashMap
]
